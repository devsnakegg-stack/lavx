lavx

A minimal, TypeScript-first, production-grade Lavalink client for Discord.js.

Philosophy: lavx is a control layer. It handles the orchestration of Lavalink nodes and Discord voice connections, leaving platform resolution to Lavalink plugins.

Setup

Installation

npm install lavx discord.js ws undici

Initialization

import { Client as DjsClient } from 'discord.js';
import { Client as LavxClient } from 'lavx';

const djs = new DjsClient({ /* intents */ });
const lavx = new LavxClient(djs, {
  nodes: [{ host: 'localhost', port: 2333, auth: 'youshallnotpass' }]
});

Core Managers

NodeMan

Handles Lavalink node lifecycles, load balancing, and failover.

PlayMan

Manages player instances across all guilds.

QMan

Persistent (or memory-based) queue management per guild.

SrcMan

Resolves queries into tracks and handles multiplatform routing.

Player API

.play() (options?)

Starts playback. Automatically resolves UnresolvedTracks.

.pause() (state: boolean)

Pauses or resumes the audio.

.setFilters() (filters: any)

Applies Lavalink filters like Equalizer or Timescale.

.setAudioOutput() ('mono' | 'stereo' | 'left' | 'right')

Configures audio channel mixing.

Queue Management

Access via player.node.client.queue.get(guildId)

Autoplay

Toggles recommendation-based playback when the queue ends.

History

Automatically keeps track of the last 100 played tracks.

Loop Modes

Support for None, Track, and Queue looping.

Event System

Listen using lavx.events.on('eventName', ... )

  • trackStart: When a track begins.
  • trackEnd: When a track finishes.
  • nodeConnect: When a Lavalink node is ready.
  • playerMove: When a player moves voice channels.
  • queueEnd: When no more tracks are available.

Example Bot

A complete implementation can be found in the /Bot folder of the repository.

// !play 
const res = await lavx.playInput(guildId, query, user);
if (res) {
    const player = lavx.play.get(guildId) || lavx.play.create({ guildId });
    if (!player.state.connected) await player.connect(voiceChannelId);
    // Track is added and started automatically
}
