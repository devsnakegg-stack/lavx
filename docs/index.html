lavx

Advanced, professional-grade Lavalink client for Discord.js.

Setup

npm install lavx discord.js ws undici

Initialize the client with your Discord.js instance and Lavalink node details.

Music Control API

.play() / .stop()

Standard playback control.

.pause() / .resume()

Pause and resume track playback.

.moveToChannel()(channelId)

Migrate the bot to a different voice channel in the same guild.

.seek() / .restart()(ms)

Change position or start the track over.

.rewind() / .forward()(ms)

Skip backward or forward by a specified amount.

.fadeIn() / .fadeOut()(ms)

Volume transition effects.

.setEQ() / .setFilters()

Full Lavalink filter support.

.clearFilters()

Reset all applied audio effects.

.balance() / .mono() / .stereo()

Channel mixing and spatial control.

.bassboost() / .nightcore() / .vaporwave()

Instant filter presets.

Advanced Queue API

.add() / .addNext()

Add tracks to the end or start of the queue.

.insert()(index, track)

Place a track at a specific position.

.jump()(index)

Instantly switch to a track in the queue.

.move() / .swap()

Reorder the upcoming tracks.

.dedupe()

Remove all duplicate tracks automatically.

.shuffle() / .clear()

Manage the entire list of tracks.

.previous()

Retrieve the last played track metadata.

Safe History System

Metadata-only storage for high performance and low memory footprint.

.get()(limit)

Retrieve up to X recent tracks.

.last() / .size()

Quick stats about your history.

.clear()

Wipe the history clean.

Smart Systems

Autoplay

Smart recommendations from YT, Spotify, Apple, Deezer.

AutoRecover

Session recovery after Lavalink or bot crashes.

AutoResume

Resume playback after temporary network disconnects.

PreloadNext

Resolve upcoming metadata before the current track ends.

Gapless

Eliminate silence between tracks for a seamless experience.

SmartBuffer

Adaptive buffering based on network stability.

Multiplatform Control

.search()(query, platform?)

Search for tracks on specific platforms with unified results.

.detectSource()(url)

Identify the platform (Spotify, YouTube, etc.) from a URL.

.fallbackSearch()

Secondary search attempt if primary resolution fails.

.bestSource()

Select the highest quality audio source available.
